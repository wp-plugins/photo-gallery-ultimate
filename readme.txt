== Description ==


An easy way to place a photo gallery with content on your Wordpress site.

###Quick Start Guide

* [photo gallery quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/photo-gallery/)

###What you get when you use the Photo Gallery Ultimate photo gallery

*  Put any image on your photo gallery
*  photo gallery headline
*  photo gallery header content
*  photo gallery footer content
*  includes your site header and footer


###Unlimited photo galleries

You can create as many photo galleries as you want.  The Photo Gallery Ultimate lets you create photo gallery and save them for when ever you are ready to use them.

###Quick Start Guide

* [photo gallery quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/photo-gallery/)



== Installation ==


1. Upload `photo-gallery` to the `/wp-content/photo-gallery/` directory
1. Activate the photo gallery through the 'Photo Gallery Ultimate' menu in WordPress
1. Click **Photo Gallery Ultimate** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The photo gallery does the programming and design.

= Is there a limit to the number of photo gallery I can create? =
No, unlimited photo gallery can be created.

= Can the photo gallery be put on my homepage? =
Yes, you can set a photo gallery as your homepage.

= Can I put any image on a photo gallery? =
Yes, any post or page can be on a photo gallery page.
